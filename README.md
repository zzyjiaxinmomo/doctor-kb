# 可丽尔医生端知识库

基于 [VitePress](https://vitepress.dev) 构建的可丽尔医生端帮助中心，提供使用指南与常见问题。

## 功能模块

- **病例管理** — 创建、查看和管理正畸病例，追踪治疗进度
- **AI报告** — 查看 AI 生成的头影测量报告和治疗方案
- **排产申请** — 提交排产和发货申请，跟踪物流状态
- **临床服务** — 预约专家面诊和远程会诊服务
- **初步方案** — 快速生成初步治疗方案，辅助临床决策
- **快速定位** — 快速定位矫治器位置
- **患者管理** — 管理患者信息、随访记录和就诊历史

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build

# 预览构建结果
pnpm preview
```

## 技术栈

- [VitePress](https://vitepress.dev) — 静态站点生成
- [Vue 3](https://vuejs.org) — 前端框架
- Node.js >= 18.0.0
