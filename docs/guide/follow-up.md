# 复诊监控

### <span style="color:#4472c4"><strong>创建复诊计划</strong></span>

当前病例已开始佩戴第一阶段矫治器后，为方便管理病例矫治进度，您可为当前病例创建复诊计划。

初始创建复诊计划时，您可选择<span style="color:#4472c4">按次数创建</span>或<span style="color:#4472c4">自定义创建</span>。

如您选择<span style="color:#4472c4">按次数创建</span>，您需要填写病例总复诊次数、每步佩戴天数，系统将按照矫治方案总步数自

动计算生成复诊计划表，您可在系统生成的基础上进行调整。

如您选择<span style="color:#4472c4">自定义创建</span>，您可按需填写定义每一次复诊安排。

复诊计划表创建后，距离预计复诊日期前三天，可丽尔将为您推送短信提醒。

您可在系统内随时调整复诊计划，仅支持调整未复诊的计划。

### <span style="color:#4472c4"><strong>添加复诊记录</strong></span>
![](/images/guide/p32-cc61fc8e07.png)

在矫治器佩戴过程中，病例进行线下复诊时，

您可在系统内为本次复诊情况进行记录。

您可依据复诊标准判断病例最近一次阶段佩戴是否符合

预期标准，下一阶段是否需要调整矫治方案。

如您线下采集了相关复诊照片资料，您也可在此处上传。

需要注意的是，如您已为当前病例创建了复诊计划，

您需要选择本次复诊记录对应的复诊计划项。

如当前病例未创建复诊计划，

您只需记录当前佩戴矫治器的步数即可。

您也可点击页面上方<span style="color:#4472c4">下载PDF</span> 。

将复诊监控文件下载使用

### <span style="color:#4472c4"><strong>登记下次复诊日期</strong></span>

在矫治器佩戴过程中，如当前病例无需创建整套复诊计划、仅某些关键节点需要复诊提醒。

您可依据实际矫治情况，在系统内为当前病例仅<span style="color:#4472c4">登记下一次复诊日期</span>。

登记完成后，预计复诊日期到期前三天，可丽尔也将为您推送病例复诊提醒。

需要注意的是，如您已为当前病例创建了整套复诊计划，调整下次复诊日期，需在复诊计划表内进行

调整。
