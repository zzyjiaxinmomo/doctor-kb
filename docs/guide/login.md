### <span style="color:#4472c4"><strong>登录方式一：账号密码登录</strong></span>

进入页面，默认为<span style="color:#4472c4"><strong>账号密码登录</strong></span>，输入手机号、密码，勾选隐私协议及服务协议后，可进行登录。

如您忘记了登录密码，可点击<span style="color:#4472c4"><strong>忘记密码</strong></span>进行密码重置。

### <span style="color:#4472c4"><strong>登录方式二：短信验证码登录</strong></span>

![](/images/guide/p05-4c806a84cb.png)

<span style="color:#c00000">需注意：</span>

<span style="color:#c00000">如您账号未绑定过微信，首次使用时，需完成账号绑定；后续使用时，无需再次绑定。</span>
