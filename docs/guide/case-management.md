# 病例管理

### <span style="color:#4472c4"><strong>病例列表</strong></span>

所有病例都将汇总至病例列表，您可在病例列表通过状态切换、重点关注来<span style="color:#4472c4">筛选</span>病例。

您可<span style="color:#4472c4">删除</span>不需要的病例草稿，<span style="color:#4472c4">一键关注</span>或<span style="color:#4472c4">取消关注</span>当前病例。

点击病例，即可进入病例详情页。

![](/images/guide/p28-5a154ca945.png)

![](/images/guide/p28-fab0c0b61b.png)

### <span style="color:#4472c4"><strong>病例详情页</strong></span>

您所管理的每位病例，都具备个人详情页。

您可在详情页实现：

查看当前病例的所有治疗方案，了解病例矫治进度，获取最新矫治器发货物流信息。

对矫治器发放进行记录，对病例全病程复诊进行监控，记录每次复诊信息。

![](/images/guide/p29-0350e9ff12.png)

如当前病例已矫治完成，或因其他情况需要暂时中止矫治进度，您可点击<span style="color:#4472c4"><strong>病例操作</strong></span>进行处理

![](/images/guide/p30-a861b6d6bb.png)

<span style="color:#4472c4"><strong>矫治器记录发放</strong></span>

当您接收到可丽尔工厂邮寄给您的矫治器后，

您可依据矫治进度为病例进行阶段发放。

为病例完成阶段矫治器发放后，

您可在系统内记录每次发放情况。

此项非必须填写项，您可按需使用。

![](/images/guide/p30-caffbef60a.png)

<span style="color:#4472c4"><strong>提交后续加工申请</strong></span>

您可通过此功能，提交病例的下一阶段矫治器加工申请。

可丽尔接收到您的申请后，

将由专业客服人员联系您进行确认。

与您确认无误后，

可丽尔工厂将进行下一阶段矫治器生产加工。

<span style="color:#4472c4"><strong>完成病例</strong></span>

![](/images/guide/p34-f415f70b77.png)

病例已经矫治完成，或因其他特殊情况需要停止矫治时，

您可在<span style="color:#4472c4"><strong>完成病例</strong></span>操作。

您需要先选择<span style="color:#4472c4">病例完成原因</span>，如病例为正常矫治完成，您需要上传相关照片资料。

如非正常矫治完成，因其他原因需要停止矫治，相关照片资料则非必须上传项。

![](/images/guide/p34-8b3a0025f0.png)

<span style="color:#4472c4"><strong>重启病例</strong></span>

病例状态为<span style="color:#4472c4">已完成</span>或者<span style="color:#4472c4">已中止</span>时，依据矫治进度安排，您可选择<span style="color:#4472c4"><strong>病例重启</strong></span>，病例重新启动后，

可丽尔将为当前病例重新设计矫治方案，生产加工佩戴矫治器。

需要注意的是，如您使用的产品是C3-Lite，则不支持病例重启操作。

选择病例重启，您需要：

确认病例基本信息、填写重启问卷、上传最新病例照片资料、提供最新模型。

![](/images/guide/p35-9460e9f6ee.png)

<span style="color:#4472c4"><strong>申请发保持器</strong></span>

病例状态为<span style="color:#4472c4">已完成</span>时，如需要病例需要保持器，您可在系统内<span style="color:#4472c4">提交保持器申请</span>。

收到您的申请后，可丽尔专业客服人员将联系您确认。

首次提交保持器申请，需要您填写下相关情况，同一病例同一方案后续再提交则无需填写。

![](/images/guide/p36-4dc94c7113.png)

需要注意的是，如当前的病例使用的产品是C3-Lite，则不支持申请发保持器。

### <span style="color:#4472c4"><strong>患者列表</strong></span>

您既往所管理的所有病例，除病例列表外，也会保存在<span style="color:#4472c4">患者列表</span>内。

您可通过年龄、性别、病例特征、是否重点关注，在全部病例数据内进行筛选。

![](/images/guide/p37-e24e8ee9e2.png)
