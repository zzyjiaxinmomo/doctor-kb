# 个人中心

### <span style="color:#4472c4"><strong>我的资料</strong></span>

首次注册账号（未提交资质认证）时，您需要进入<span style="color:#4472c4">个人中心</span>内<span style="color:#4472c4">我的资料</span>填写个人信息。

资质认证通过后，您可随时进入<span style="color:#4472c4">我的资料</span>页修改您的个人资料。

您可修改以下内容：

您的账号头像、姓名、邮箱号、接收通知手机号码、系统登录手机号码、个人职称、证件号码、当前

执业机构。

需要注意的是，您的资质审核通过后，将不支持修改证件照片。

如您需要使用微信扫码登录及微信消息提醒推送，您可在此页面，扫码绑定您的微信。

![](/images/guide/p39-0d90dbc9b7.png)

### <span style="color:#4472c4"><strong>我的机构</strong></span>

注册账号后，您完善个人资料后，需要在<span style="color:#4472c4">我的机构</span>内维护您的执业点信息。

执业点是管理病例的必要前提条件，如您未维护执业点信息，后续将无法添加和管理病例。

如您有多个线下执业点，您可在系统内进行维护，首次添加后，可随时进行修改。

维护执业点需要您填写：

执业点名称、执业点地址，归属科室

![](/images/guide/p40-8f89ff2aa8.png)

### <span style="color:#4472c4"><strong>收货地址管理</strong></span>

收货地址是提交和管理病例、线下成功接收矫治器的必要前提条件，如您未维护收货地址，后续将无

法提交病例。

如您有多个线下收货地址，您可在系统内进行维护，首次添加后，可随时进行修改。

您需要将某一个常用的地址设置为您的默认地址，流程操作中，系统将选择您的默认地址，您可切换

至其他地址。

维护收货地址需要您填写：

联系人姓名、联系方式、具体地址信息

![](/images/guide/p41-66d86c6a95.png)

### <span style="color:#4472c4"><strong>临床偏好设置</strong></span>

为了更准确地提供矫治治疗方案，您可在<span style="color:#4472c4">个人中心</span>的<span style="color:#4472c4">临床偏好设置</span>内设置您的个人偏好。

可丽尔专业设计出具方案时，将充分参考您的个人偏好。

![](/images/guide/p42-7d14cc6816.png)

### <span style="color:#4472c4"><strong>我的评价</strong></span>

您在审核可丽尔治疗方案后，如对当前治疗方案填写了评价意见。

您可在<span style="color:#4472c4">个人中心</span>的<span style="color:#4472c4">我的评价</span>内，查看所有您的评价意见。

![](/images/guide/p43-a58e908122.png)
