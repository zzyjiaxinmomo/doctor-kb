# 工作台

### <span style="color:#4472c4"><strong>资质认证</strong></span>

如您还未提交资质认证，登录后会为您展示可丽尔病例操作流程。

您可点击<span style="color:#4472c4"><strong>立即认证</strong></span>，完成资质认证。资质认证后，您可创建管理您的病例。

![](/images/guide/p07-8a2c5b9eda.png)

完成资质认证，您需要提供：<span style="color:#4472c4">医师执业证</span>照片、<span style="color:#4472c4">医师资格证</span>照片、<span style="color:#4472c4">医师资格证编号</span>

提交后等待后台工作人员审核，审核通过即完成资质认证。

维护完成您的资质信息后，您还需要进入<span style="color:#4472c4"><strong>我的机构</strong></span>维护您的执业点信息。

进入<span style="color:#4472c4"><strong>收货地址管理</strong></span>，维护您的收货地址。

以上内容操作，详见<span style="color:#4472c4">个人中心-</span><span style="color:#4472c4"> </span><span style="color:#4472c4">我的资料、我的机构、我的收货地址</span>

如您已完成资质认证，您可通过<span style="color:#4472c4">工作台</span>实现以下操作：


### 1、通过病例姓名、病例号快捷<span style="color:#4472c4">搜索病例</span>。

### 2、一键<span style="color:#4472c4">切换</span>您的执业机构。
![](/images/guide/p08-22f0fb11f6.png)


### 3、查看<span style="color:#4472c4">站内信通知</span>。

![](/images/guide/p08-3b1069e2f8.png)


### 4、点击头像可进入<span style="color:#4472c4">个人中心、修改密码</span>。
![](/images/guide/p08-6be4d3cf61.png)

### 5、下载治疗同意书（成人/儿童）。

### 6、下载ClearCheck 软件安装包。

### 7、下载app。

### 8、快捷查看当前<span style="color:#4472c4">病例总量</span>。
![](/images/guide/p08-4d8f2287e3.png)

点击栏目即跳转对应病例列表。


### 9、快捷<span style="color:#4472c4">预约顺丰快递</span>，邮寄实物模型。
![](/images/guide/p08-4197e394f3.png)

### 10、下载<span style="color:#4472c4">帮助文档</span>。


### 11、快捷处理<span style="color:#4472c4">待审核治疗方案</span>

所有需要您审核的方案都将展示在工作台。

文字方案可直接处理。

动画方案将跳转ClearCheck 软件查看。
![](/images/guide/p08-7ca5ea5c20.png)


### 10、快捷处理<span style="color:#4472c4">待完善资料</span>

如您提交的病例资料或模型未通过质检。

系统将会反馈您不通过的病例及不合格原因，您可再次<span style="color:#4472c4">完善资料</span>或<span style="color:#4472c4">提交模型</span>。

![](/images/guide/p09-47b6573f7a.png)

![](/images/guide/p09-a08730c080.png)

### 11、快捷处理<span style="color:#4472c4">矫治器通知</span>

如您的病例阶段佩戴矫治器即将到期，系统将会为您展示这些病例。

您可一键提交<span style="color:#4472c4">后续矫治器加工申请</span>。

![](/images/guide/p09-4418e2e8ce.png)

![](/images/guide/p09-c053c4e0d2.png)

### <span style="color:#4472c4"><strong>文字方案处理流程</strong></span>

如可丽尔专业设计师为您提供的是无动画方案，即文字方案。

![](/images/guide/p10-4ce73a82c2.png)

您将在病例方案卡片上看到<span style="color:#4472c4"><strong>无动画</strong></span>标记。

操作鼠标悬浮至对应方案

出现<span style="color:#4472c4">查看文字方案</span>操作按钮。

![](/images/guide/p10-b90eb33a51.png)

点击后，当前页面弹窗，可查看本次方案。

如您认可当前方案，

可点击右上角<span style="color:#4472c4">批准</span>按钮，批准本次方案。

如您对当前方案有意见或建议，

您可在右侧输入框内填写提交您的修改意见。

![](/images/guide/p10-83c4ddbff0.png)

您在左上角页签，切换<span style="color:#4472c4">意见历史</span>。

查看当前病例的所有历史方案及修改意见。
