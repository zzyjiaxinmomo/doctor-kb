# 新建初诊病例

### 1. <span style="color:#4472c4"><strong>创建入口</strong></span>

点击工作台或病例列表的入口<span style="color:#4472c4"><strong>新建病例</strong></span>进入病例创建流程。

![](/images/guide/p11-075dc88843.png)

![](/images/guide/p11-e07a5ff887.png)

### 2. <span style="color:#4472c4"><strong>病例基本信息</strong></span>

新建病例，首先需填写病例基本信息：<span style="color:#4472c4">姓名</span>、<span style="color:#4472c4">性别</span>、<span style="color:#4472c4">出生日期</span>、<span style="color:#4472c4">就诊机构。</span>

机构地址无需填写，依据您选择的就诊机构自动匹配展示。

如当前病例为您重点关注，您可点击☆进行标记。

如您对当前病例有其他备注需要记录，可在患者备注内填写。此处的备注仅供您本人使用。

![](/images/guide/p11-4951ad4b0a.png)

### 3. <span style="color:#4472c4"><strong>病例资料收集</strong></span>

基本信息填写后，需要您上传患者本次就诊的相关照片资料。

照片资料包含以下内容：

![](/images/guide/p12-6d0b748433.png)

#### <span style="color:#4472c4"><strong>面部照：</strong></span>

侧面照、正面照、正面露齿微笑照、

正面45 度照。

#### <span style="color:#4472c4"><strong>口腔内照片：</strong></span>

上牙弓、覆盖照、下牙弓、右面咬合、

正面咬合、左面咬合。

#### <span style="color:#4472c4"><strong>影像资料：</strong></span>

全颌曲面断层片、头颅侧位定位片。

#### <span style="color:#4472c4"><strong>其他图像资料：</strong></span>

本项非必须项，如您有其他照片资料需要补充，可在此处上传。

#### <span style="color:#4472c4"><strong>其他视频资料：</strong></span>

本项非必须项，如您有其他照片资料需要补充，可在此处上传。

<span style="color:#c00000">为方便操作，您可以一次性将所需照片上传至页面右侧照片池中，从照片池内选择照片拖拽</span>

<span style="color:#c00000">至对应位置即可。</span>

已上传的图片，支持：上下翻转、左右翻转、裁剪、向左旋转、向右旋转。

#### <span style="color:#4472c4"><strong>照片拍摄标准</strong></span>

为了向您提供最准确的病例矫正治疗方案，请您按以下标准提供病例相关照片资料。

#### <span style="color:#4472c4"><strong>口腔摄影器材选择及拍摄数据设置：</strong></span>

最佳为单反相机配合100mm 微距镜头，闪光灯使用外置环闪

1、面像拍摄相机设置：M 档、ISO100、1/60 秒、F4.0-10.0

2、口内像拍摄相机设置：M 档、ISO100、1/125 秒、F25-32

#### <span style="color:#4472c4"><strong>面部照</strong></span>

#### <span style="color:#4472c4"><strong>正面放松照及微笑照</strong></span>

1、患者身体坐端，背部挺直，面部呈自然放松状态，两眼平视前方。

2、拍摄者要与患者同一高度。

3、牙齿自然咬合在牙尖交错位，唇自然放松。

4、需要转换角度时，拍摄者尽量保持原位置不变，请患者配合转动身体。

5、头面部比例大小：头面部应占画面70％。

6、拍摄微笑照时，指导患者在牙尖交错咬合状态下自然充分微笑，便于观察牙列中线。

整理发型，去除影响视觉的耳环首饰，暴露额头、双耳。

#### <span style="color:#4472c4"><strong>侧面90 度照</strong></span>

1、自然头位，双眼直视前方，能看到对侧睫毛为最佳。

2、牙齿自然咬合在牙尖交错位，唇自然放松。

#### <span style="color:#4472c4"><strong>侧面45 度照</strong></span>

1、自然头位，双眼直视前方，鼻尖在对侧面颊部边缘以内为最佳。

2、牙齿自然咬合在牙尖交错位，唇自然放松。

#### <span style="color:#4472c4"><strong>口内照</strong></span>

#### <span style="color:#4472c4"><strong>口内正面像</strong></span>

1、口角拉钩手柄中央的位置与咬合平面位于一条直线上，让拉钩在画面的四角上均等。

2、将上颌中线位于图像中央（纵向）。

3、将合平面与镜头线呈水平（横向）。

4、使左右颊粘膜间隙均等。

#### <span style="color:#4472c4"><strong>口内左（右）侧位照</strong></span>

1、口角拉钩左右手柄中心与咬合平面在一条线上。

2、前牙侧口角拉钩（非摄影侧）与前牙不接触。

3、磨牙侧口角拉钩（摄影侧）尽可能打开。

4、患者尖牙位置放置在图像中央等，从正中到左右的牙齿数相等。

#### <span style="color:#4472c4"><strong>上（下）颌咬合面照</strong></span>

1、反光镜离开最后磨牙，不要接触。

2、上、下颌面应位于画面中央，上中切牙在画面两侧边缘距离相等，画面的视觉效果相当于相机

垂直颌面拍照一样。

3、正中线位于反光镜中央。反光镜边缘与左右磨牙的间隙均等。反光镜开口角度尽可能大（能观

察到前牙区的舌侧）。拍摄下颌时，舌头卷至镜子后方。

### 4. <span style="color:#4472c4"><strong>产品选择</strong></span>

您需要选择当前病例全周期矫正使用的治疗产品。

依据使用人群及产品特点，分为<span style="color:#4472c4">儿童产品</span>及<span style="color:#4472c4">成人产品</span>两类，您可按实际病例情况及需求选择。

同一病例仅可选择一项产品进行矫治治疗。

![](/images/guide/p15-719285bab1.png)

![](/images/guide/p15-41d2e41047.png)

### 5. <span style="color:#4472c4"><strong>临床诊断填写</strong></span>

依据当前病例的情况及您选择的治疗产品，需要填写临床诊断。

![](/images/guide/p16-0be9cd7437.png)

临床诊断依据治疗产品有所区分。

页面标记<span style="color:#c00000">*必填</span>的为必须填写项。

其余为非必须项。

为方便您操作，您可在左侧目录中进行选择。

点击后，中间页面会自动定位至您选中的项目。

您可点击右侧图像参考区域中的照片。

随时类比病例图像资料完成诊断填写。

### 6. <span style="color:#4472c4"><strong>治疗处方填写</strong></span>

依据当前病例的情况及您选择的治疗产品，需要填写治疗处方。

![](/images/guide/p17-deecae2d9f.png)

治疗处方依据治疗产品有所区分。

页面标记<span style="color:#c00000">*必填</span>的为必须填写项。

其余为非必须项。

为方便您操作，您可在左侧目录中进行选择。

点击后，中间页面会自动定位至您选中的项目。

您可点击右侧图像参考区域中的照片。

随时类比病例图像资料完成治疗处方填写。

### 7. <span style="color:#4472c4"><strong>病例模型上传/邮寄</strong></span>

#### <span style="color:#4472c4"><strong>数字模型</strong></span>

如您为当前病例采集的数字口扫模型，您需要将相关数字模型文件进行上传。

其中<span style="color:#4472c4">上颌</span>及<span style="color:#4472c4">下颌</span>口扫数字文件是必须上传项。

您可将上颌、下颌、咬合记录打包为压缩包，统一使用<span style="color:#4472c4">打包上传</span>进行上传。

如您有其他文件资料需要一并上传，您可在<span style="color:#4472c4">其他内容</span>内上传。

![](/images/guide/p18-cb213fd499.png)

#### <span style="color:#4472c4"><strong>隐形矫治口扫模型合格标准</strong></span>

1、牙冠形态扫描完整（近中、远中、颌面、唇侧、颊侧、舌侧）。

2、上颌、下颌记录完整。

3、牙龈正常的矫治患者，牙龈边缘需清晰、连续、完整。

#### <span style="color:#4472c4"><strong>实物模型</strong></span>

如您为当前病例采集的<span style="color:#4472c4">实物模型（硅胶或石膏）</span>，您需要将实物模型邮寄至可丽尔中心。

可丽尔中心收到快递后，将进行专业质检。

如您已自行下单，可按需填写快递单号；

您可通过<span style="color:#4472c4"><strong>顺丰快捷下单</strong></span>入口，填写物流订单信息后，将由顺丰快递员联系您并进行上门取件。

需要注意的是：

如您选择的治疗产品是C4-flex，也需要上传CBCT 文件。

依据顺丰物流规定，顺丰快递员上门取件前，您可随时取消本次快递订单。

如顺丰快递员已上门完成取件服务，您将无法操作取消订单。

如取件后有任何需要帮助的地方，您可随时联系您的客户经理或可丽尔专业客服人员进行协助处理。

<span style="color:#c00000">实物模型顺丰邮寄服务完全免费，快递费用由可丽尔承担，您无需为此支付任何费用。</span>

#### <span style="color:#4472c4"><strong>实物模型（硅胶或石膏）取模标准及步骤说明</strong></span>

#### <span style="color:#4472c4"><strong>合格硅橡胶重要性说明</strong></span>

对于我司隐形矫治器非常重要的生产基础工作，就是医生给我司提供合格的患者全口模型，只有取得

清晰准确的印模，才能制作出符合人体生理要求的矫治器；它的质量好坏直接影响后续的方案设计、

患者能否舒适的戴上矫治器。

不合格模型导致后果，如下说明：

矫治器无法就位。矫治器局部不贴合。矫治过程不能按照预期目标进行。

矫治周期延长。降低患者矫治成功信心。

#### <span style="color:#4472c4"><strong>几种特殊情况印模的取制</strong></span>

1、对于牙周炎导致临床冠较长的患者，建议填倒凹后，再取硅橡胶模型。

2、对于某些需要牙体治疗的牙齿，建议完善牙体治疗之后，再取硅橡胶模型。

3、第三磨牙萌出且建立正常咬合，不需拔出智齿，硅橡胶模型至少应确保至第三磨牙近中1/2 以上。

#### <span style="color:#4472c4"><strong>重取操作说明</strong></span>

1、对于如上不合格模型，请尽快重取，并邮寄，我们将尽快处理，避免矫治计划的延误或反弹。

2、取制完成后，请初步检查（是否符合合格标准），以避免重复取模或矫治时间延长。

3、若操作中存在需要沟通项目，请直接联系客服人员。
